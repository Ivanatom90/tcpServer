# tcpServer
Порядок работы с tcpServer

1) Запускаем приложение tcpServer
2) Дожидаемся подключения клиентов. "tcpClient". контролировать подключение клиентов можно во вкладке logs графичиской формы.
3) Во вкладке clients settings выставляем ограничения по метрикам Metrics bourder, вводим значение метрики при меньшем значении которой будут высылаться предупреждения в Log файле от клиента. Нажимаем кнопку Send;
4) Нажимаем кнопку Start для запуска отправки сообщейний от клиентов.
5) Нажатием кнопки Stop можно остановть рассылку информации от клиентов.
